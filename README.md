# Codebase Investigator

A production-grade AI codebase investigator that uses a dual-agent architecture to ensure trustworthy, grounded answers. 

Built with **Next.js 15 (App Router)**, **TypeScript**, and the **Google Gemini API**.

## 🌟 Key Features

### 1. Independent Audit Architecture (The Hard Rule)
Every answer generated by the Investigator (Gemini Flash) is independently audited by a separate Auditor agent (Gemini Pro). The auditor runs in a **completely fresh context** with a different system prompt, ensuring zero self-scoring or context pollution.

### 2. Programmatic Citation Verification
When the Investigator cites code (e.g., `[src/auth.ts:L10-L20]`), the system deterministically checks:
- Does the file exist in the repo?
- Do the line numbers exist?
This is done programmatically *before* the AI audit, feeding hard facts into the Auditor.

### 3. Coherent Multi-Turn Memory
The system maintains coherence across 8-15 turns using:
- Full conversation history (for context)
- **Claim Tracking**: Factual claims are extracted from every answer and checked against future answers to catch contradictions.

### 4. Agentic Tool Belt
The Investigator uses Gemini Function Calling to explore the codebase:
- `list_structure`: Understand repo layout
- `search_files`: Find files by name/glob
- `grep_code`: Regex search across the repo
- `read_file`: Read specific lines of a file

### 5. Premium UI
- Glassmorphism dark mode
- Inline file viewer with syntax highlighting and citation line highlighting
- Expandable Audit Cards
- Streaming Server-Sent Events (SSE) for real-time tool activity

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the app:**
   Navigate to [http://localhost:3000](http://localhost:3000).

4. **API Key:**
   The app will prompt you for a Gemini API Key on the first screen. The key is stored safely in your browser's `localStorage` and is never saved on the server.

## Architecture

- `src/lib/agent/investigator.ts`: The main agentic loop that decides which tools to call.
- `src/lib/agent/auditor.ts`: The independent reviewer that evaluates the final answer.
- `src/lib/citations/verifier.ts`: Programmatic verification of file and line existence.
- `src/app/api/chat/route.ts`: Streaming API endpoint orchestrating the pipeline.
